# @dodamind/eslint-config-dodamind

ESLint preset for dodamind projects.

## Install

```sh
npm install @dodamind/eslint-config-dodamind --save-dev
```
## Usage
### .eslintrc.cjs
```js
module.exports = {
  extends: [ 
    '@dodamind/eslint-config-dodamind',
    // ...
  ],
  rules: {
    // ...
  },
}
```
